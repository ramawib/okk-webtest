TENTANG OKK UI 2021

Orientasi Kehidupan Kampus Universitas Indonesia 2021

OKK UI 2021 hadir sebagai akses utama bagi mahasiswa untuk mengawali gerbang perkuliahan. Lebih dari 8000 mahasiswa UI 2021 dengan keberagamannya masing-masing akan berkumpul dalam satu momen, satu semangat, dan satu kebanggaan sebagai perayaan atas pencapaiannya menyandang status mahasiswa UI.

OKK UI menunggu kontribusi terbaik kalian untuk UI dan Indonesia. Terimalah persembahan hangat dari kami, OKK UI 2021.
